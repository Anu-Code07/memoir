## Installation Overview

<Steps>
  <Step title="Install the Memoir CLI">
    Our CLI handles project setup, development, and deployment.

    ```bash
    brew install memoir/tap/cli
    ```

    [Installation Guide →](/getting-started/installation)
  </Step>
  <Step title="Initialize Your Project">
    Run `memoir init` to scaffold a new project with Convex backend configured.

    ```bash
    memoir init my-project
    cd my-project
    ```

    [5-Minute Quickstart →](/getting-started/five-minute-quickstart)
  </Step>
  <Step title="Configure for Your Stack">
    Set up authentication, customize settings, and connect your AI provider.

    ```bash
    memoir dev
    ```

    [Configuration Guide →](/getting-started/configuration)
  </Step>
</Steps>
