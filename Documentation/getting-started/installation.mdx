# Installation

Get Memoir up and running in minutes.

## Quick Start

The fastest way to get started with Memoir:

<Steps>
  <Step title="Install the CLI">
    Install Memoir globally:
    
    <Tabs groupId="install-method">
      <TabItem value="brew" label="Homebrew (Recommended)">
        <Terminal>brew install memoir/tap/cli</Terminal>
      </TabItem>
      <TabItem value="npm" label="npm">
        <Terminal>npm install -g @getmemoir/cli</Terminal>
      </TabItem>
    </Tabs>
  </Step>
  <Step title="Verify installation">
    Check that the CLI is installed correctly:
    
    <Terminal>memoir --version</Terminal>
    
    Should output: `0.27.4` or higher
  </Step>
  <Step title="Create your first project">
    Initialize a new Memoir project:
    
    <Terminal>memoir init my-memoir-agent</Terminal>
    
    The interactive wizard will guide you through setup in under 5 minutes.
  </Step>
</Steps>

## Prerequisites

<Callout type="warning" title="Requirements">
Before installing Memoir, ensure you have:

**Required:**
- **Node.js >= 20** — Download from [nodejs.org](https://nodejs.org)
- **npm or pnpm** — Comes with Node.js
</Callout>

Check your versions:

<Terminal>node --version && npm --version</Terminal>

Expected output: Node.js `v20.0.0+` and npm `9.0.0+`

<Accordion>
  <AccordionItem title="Optional Dependencies">
    **Convex Account** — For cloud deployments (free tier available)
    - Sign up at [convex.dev](https://convex.dev)
    - Required for vector search features
    - Not needed for local development

    **Docker Desktop** — For graph database features
    - Download from [docker.com](https://www.docker.com/products/docker-desktop)
    - Only needed for advanced graph queries
    - Not required for basic Memoir usage

    **OpenAI API Key** — For embeddings
    - Get from [platform.openai.com](https://platform.openai.com)
    - Or use any other embedding provider
    - Memoir works without embeddings (keyword search)
  </AccordionItem>
</Accordion>

## Installation Methods

### Method 1: Global Installation (Recommended)

Install the CLI globally to use it from anywhere:

<Tabs groupId="package-manager">
  <TabItem value="brew" label="Homebrew (Recommended)">
    <Terminal>brew install memoir/tap/cli</Terminal>
    
    Homebrew handles PATH configuration automatically and makes updates easy with `brew upgrade`.
  </TabItem>
  <TabItem value="npm" label="npm">
    <Terminal>npm install -g @getmemoir/cli</Terminal>
  </TabItem>
  <TabItem value="yarn" label="yarn">
    <Terminal>yarn global add @getmemoir/cli</Terminal>
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    <Terminal>pnpm add -g @getmemoir/cli</Terminal>
  </TabItem>
</Tabs>

**Verify installation:**

<Terminal>memoir --version</Terminal>

**Now create projects anywhere:**

<Terminal>memoir init my-agent && memoir start</Terminal>

The CLI uses persistent config with absolute paths, so `cd` is not required.

### Method 2: Using npx (No Installation)

Use the CLI without installing it globally:

<Terminal>npx @getmemoir/cli init my-agent</Terminal>

<Callout type="info" title="When to use npx">
- One-time project creation
- Testing before global install
- CI/CD environments

**Trade-off:** Slightly slower (downloads each time)
</Callout>

### Method 3: Project-Specific Installation

Install as a dev dependency in your project:

<Tabs groupId="package-manager">
  <TabItem value="npm" label="npm">
    <Terminal>npm install --save-dev @getmemoir/cli</Terminal>
  </TabItem>
  <TabItem value="yarn" label="yarn">
    <Terminal>yarn add --dev @getmemoir/cli</Terminal>
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    <Terminal>pnpm add -D @getmemoir/cli</Terminal>
  </TabItem>
</Tabs>

**Add to package.json:**

<CodeBlock filename="package.json" language="json">
{`{
  "scripts": {
    "memoir": "memoir",
    "dev": "memoir dev",
    "start": "memoir start",
    "stop": "memoir stop"
  }
}`}
</CodeBlock>

**Then run:**

<Terminal>npm run memoir init && npm run dev</Terminal>

## Verification

After installation, verify everything works:

<Steps>
  <Step title="Check CLI Installation">
    <Terminal>memoir --version</Terminal>
    
    Should show: `0.27.4` or higher
    
    <Terminal>memoir --help</Terminal>
    
    Should display available commands
  </Step>
  <Step title="Test CLI Connection">
    <Terminal>memoir init test-project</Terminal>
    
    Start services (from any directory):
    
    <Terminal>memoir start</Terminal>
    
    Check status:
    
    <Terminal>memoir status</Terminal>
  </Step>
  <Step title="Verify Convex Dashboard">
    **For local development:** Open http://127.0.0.1:3210
    
    **For cloud deployment:**
    
    <Terminal>memoir convex dashboard</Terminal>
    
    Opens your deployment in browser
  </Step>
</Steps>

## Troubleshooting

<Accordion>
  <AccordionItem title="memoir: command not found" defaultOpen={true}>
    **Cause:** CLI not installed or not in PATH

    **Fix (Recommended):**

    <Terminal>brew install memoir/tap/cli</Terminal>

    **Alternative (npm):**

    <Terminal>npm install -g @getmemoir/cli</Terminal>

    If using npm, check global bin path is in your PATH:

    <Terminal>npm config get prefix</Terminal>
  </AccordionItem>

  <AccordionItem title="memoir init fails">
    **Possible causes:**
    - Node.js version &lt; 20
    - No internet connection
    - npm permissions

    **Fix (Recommended):**

    Update Node.js to v20+, then reinstall via Homebrew:

    <Terminal>brew reinstall memoir/tap/cli</Terminal>

    **Alternative (npm):**

    <Terminal>npm cache clean --force && npm install -g @getmemoir/cli</Terminal>
  </AccordionItem>

  <AccordionItem title="Cannot find module @getmemoir/sdk">
    **Cause:** Dependencies not installed in project

    **Fix:**

    <Terminal>cd your-project && npm install</Terminal>
  </AccordionItem>

  <AccordionItem title="Convex connection failed">
    **Cause:** Convex server not running

    **Fix:**

    <Terminal>memoir start</Terminal>

    If issues persist, restart:

    <Terminal>memoir stop && memoir start</Terminal>
  </AccordionItem>

  <AccordionItem title="Port Conflicts">
    **Cause:** Another Convex instance on same port

    **Fix:** Use interactive dev mode:

    <Terminal>memoir dev</Terminal>

    Press `k` to kill conflicting processes.
  </AccordionItem>
</Accordion>

## Upgrading

To upgrade to the latest version:

<Tabs groupId="package-manager">
  <TabItem value="brew" label="Homebrew (Recommended)">
    <Terminal>brew upgrade memoir/tap/cli</Terminal>
  </TabItem>
  <TabItem value="npm" label="npm">
    <Terminal>npm install -g @getmemoir/cli@latest</Terminal>
  </TabItem>
  <TabItem value="yarn" label="yarn">
    <Terminal>yarn global upgrade @getmemoir/cli</Terminal>
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    <Terminal>pnpm add -g @getmemoir/cli@latest</Terminal>
  </TabItem>
</Tabs>

Verify new version:

<Terminal>memoir --version</Terminal>

Update SDK and Convex in your projects:

<Terminal>memoir update</Terminal>

<Callout type="info">
The `memoir update` command will:
- Check for newer versions of `@getmemoir/sdk`
- Check for newer versions of `convex`
- Offer to update all projects
- Redeploy backend if needed

**Note:** The CLI stores project paths in its config, so you can run `memoir update` from any directory.
</Callout>

## Uninstalling

If you need to uninstall:

<Tabs groupId="package-manager">
  <TabItem value="npm" label="npm">
    <Terminal>npm uninstall -g @getmemoir/cli</Terminal>
  </TabItem>
  <TabItem value="yarn" label="yarn">
    <Terminal>yarn global remove @getmemoir/cli</Terminal>
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    <Terminal>pnpm remove -g @getmemoir/cli</Terminal>
  </TabItem>
</Tabs>

Remove from a project:

<Terminal>cd your-project && npm uninstall @getmemoir/sdk convex</Terminal>

Clean up configuration (optional):

<Terminal>rm ~/.memoirrc</Terminal>
